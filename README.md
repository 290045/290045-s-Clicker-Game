# 🕳️ 290045's Clicker Game

A space-themed, incremental idle clicker game built from scratch using pure HTML, CSS, and vanilla JavaScript. Harvest dark matter, collapse space-time, and expand your cosmic power directly from your web browser.

## 🚀 Live Demo
Play the game live right here: **https://290045.github.io/290045-s-Clicker-Game/**

---

## 🌌 Features
- **Cosmic Theme:** Stylized deep-space void theme using neon violet indicators and terminal fonts.
- **Interactive Mechanics:** Tap the central event horizon to generate Dark Matter particles.
- **Singularity Upgrades:** Spend your harvested matter to automate production with Mini Singularities.
- **Scaling Economy:** Upgrade costs scale automatically as your cosmic empire expands.
- **Zero Dependencies:** Formatted purely with standard HTML5, CSS3, and JavaScript—no frameworks required.

---

## 🛠️ How to Play
1. **Harvest:** Click on the central Black Hole (`🕳️`) to gather Dark Matter.
2. **Automate:** Once you reach the target threshold, purchase a **Mini Singularity** to start generating Dark Matter automatically every second.
3. **Scale:** Keep buying upgrades to multiply your production and see how much matter you can compress into the void!

---

## 💻 Tech Stack
- **Structure:** HTML5
- **Styling:** CSS3 (Flexbox, Drop-Shadow Effects, Active Scaling States)
- **Logic:** JavaScript (DOM Manipulation, setInterval Event Loops)
- **Hosting:** GitHub Pages
